Основные понятия:

Promise: 
Это объект, который представляет собой обещание (promise) того, что у нас в будущем будет результат (успешный или ошибка).
Он может быть в состоянии: pending (ожидание), fulfilled (выполнено), rejected (отклонено).	
Квитанция из гардероба или номерок из кафе. 
Это обещание получить свою вещь/еду.	hlzApi.apiHlzControllerUpdate(...) возвращает Promise с ответом от сервера.


async	Ключевое слово, которое ставится перед объявлением функции.
Оно говорит: "Эта функция всегда возвращает Promise". 
Даже если внутри она просто возвращает число, это число будет обёрнуто в Promise.	
Ты говоришь: "Я асинхронный работник, я всегда даю обещание (Promise) сделать работу".	
async ({}, use) => { ... } — фикстура объявлена как async, потому что внутри есть await.


await	Ключевое слово, которое можно использовать только внутри async-функций. 
Оно ставится перед Promise.
Оно говорит: "Приостанови выполнение этой функции до тех пор, пока Promise не будет выполнено или отклонено. 
Верни результат."	Ты показываешь номерок официанту и ждёшь (await), пока он не принесёт твой заказ.
Ты не делаешь ничего другого, пока не получишь его.	
const { data } = await createTestHlz(request); — код ждёт, пока запрос не выполнится,
и только потом записывает результат в переменную data.
